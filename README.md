# Manuel Escalera — Portfolio

Personal portfolio built with Astro, React and TypeScript. Live at [manuelescalera.dev](https://manuelescalera.dev).

## Stack

- **Astro 5** — static site generation (SSG)
- **React 18** + **TypeScript** — interactive islands
- **Three.js** + `@react-three/fiber` — 3D voxel art model viewer
- **Zustand** — shared state across islands (dark mode, language, loader)
- **CSS Modules** + CSS variables — component-scoped styles

## Features

- Dark / light mode with FOUC prevention
- Bilingual (ES / EN) with language detection
- 3D interactive model with OrbitControls
- Fully responsive
- Deployed on Cloudflare Pages

## Dev

```bash
npm install
npm run dev
npm run build
```
